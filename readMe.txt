本文件夹包括如下文件：
myPython.zip  ――包括把node架构文件合并之前的所有代码，现在已经不是运行必须文件，放在这里作为我们劳动过程的证明
mypython-complete.js  ――node架构 文件集成以后的JS代码，有1W+行
myPython解释器展示.pptx  ――展示所用的PPT
python解释器.html  ――运行主网页，在此网页里输入代码和得到运行结果
测试用例.txt  ――两个能运行的测试用例和输出结果，还有中间传递的代码
说明文档.docx  ――具体说明此语言的语法结构等
语法树1.png  ――测试用例1的语法树
语法树2.png  ――测试用例2的语法树
 
运行时打开“python解释器.html”，在“源代码”处的文本框输入代码，然后按下“运行提交”按钮，可在“运行结果”下面的文本框内得到输出结果
